# Где искать стажировку и junior-работу

## Основные направления для Ивана

1. Python Intern / Junior Python Developer.
2. Data Analyst Intern.
3. Junior QA / QA Intern.
4. QA Automation Intern на Python.
5. Technical Support Engineer / Support Developer — только если есть перспектива перейти в разработку.
6. Преподаватель программирования/математики — как параллельный доход.

## Компании и программы

Проверяй наборы регулярно: стажировки часто открываются волнами.

- Т-Банк / Т-Старт: разработка, QA, аналитика, IT-аналитика.
- Яндекс / Young&&Yandex: backend, frontend, testing, ML, DevOps, аналитика.
- Сбер / SberStudent / SberSeasons: IT и аналитические направления.
- VK стажировки: разработка, аналитика, тестирование.
- Ozon Tech / internships / Route 256: backend, аналитика, QA, engineering.
- Avito Tech: стажировки и образовательные программы.
- МТС, Альфа-Банк, Совкомбанк Технологии, Лаборатория Касперского, Positive Technologies, Контур.

## Как откликаться

Сопроводительное письмо должно быть коротким:

```text
Здравствуйте. Я студент 3 курса ННГУ, направление «Фундаментальная информатика и информационные технологии». Ищу стажировку/начальную позицию от 20 часов в неделю. Умею Python, SQL, Git, работал с NumPy, pandas, Flask, MATLAB. В портфолио есть проект по обработке SEM-изображений фоторезиста с расчётом CD/LER/LWR, а также проекты по Flask, SQL и алгоритмам. Готов выполнить тестовое задание.
```
