# Отбор работ для GitHub

Источник: архив `Для гитхаба.zip`.

## Что включено в эту папку

1. `01_lithography_real_sem_analysis` — сильный основной проект. Оставить в портфолио.
2. `02_lithography_synthetic_metrics` — сильный основной/научный проект. Оставить в портфолио.
3. `03_restaurant_booking_web_app` — хороший web+DB проект, но перед публикацией нужна доработка схемы БД и инструкции запуска.
4. `04_numerical_methods_notebooks` — учебный репозиторий по вычислительным методам, использовать как дополнительный.
5. `05_maze_pathfinding_needs_refactor` — алгоритмический проект, использовать только после рефакторинга.
6. `06_sql_airline_queries` — маленький SQL-проект, использовать как дополнение.
7. `07_csharp_desktop_apps_reference` — C# WinForms-проекты, использовать только если нужен .NET/C# трек.

## Что не включено

- `venv`, `__pycache__`, `.vs`, `bin`, `obj`, `*.pyc`, `*.exe`, `*.dll`, `*.pdb`;
- дубли файлов;
- простые HTML-страницы `hello_world`, старые простые сайты;
- офисные временные файлы `~$...`;
- методички и учебники;
- большинство отчетов по физике/ОТЦ/акустике/вакуумной электронике;
- Mathcad/Kompas-файлы как основной материал для IT-GitHub.

## Главная рекомендация

В резюме делать акцент на Python, анализ данных/изображений, численные методы, SQL и преподавательский опыт. Главным проектом поставить обработку SEM-изображений фоторезиста, вторым — модельные метрики литографии, третьим — Flask/DB web app после доработки.
