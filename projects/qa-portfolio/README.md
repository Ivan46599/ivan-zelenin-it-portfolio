# QA Portfolio

Набор QA-документов для демонстрации базовых навыков manual testing: чек-листы, тест-кейсы, баг-репорты и API-запросы.

## Состав

- `checklists/restaurant_booking_checklist.md` — чек-лист web-приложения бронирования;
- `test_cases/restaurant_booking_test_cases.md` — тест-кейсы;
- `bug_reports/sample_bug_reports.md` — примеры баг-репортов;
- `postman/restaurant_booking_collection.json` — пример Postman collection.

## Что показывает проект

Проект показывает умение формализовать проверки: предусловия, шаги, ожидаемый результат, приоритет, серьёзность дефекта и API-запросы.
