# QA Portfolio

Набор QA-документов для откликов на Junior QA / QA Intern. Можно использовать как демонстрацию понимания тест-дизайна, баг-репортов и API-тестирования.

## Состав

- `checklists/restaurant_booking_checklist.md` — чек-лист web-приложения бронирования.
- `test_cases/restaurant_booking_test_cases.md` — тест-кейсы.
- `bug_reports/sample_bug_reports.md` — примеры баг-репортов.
- `postman/restaurant_booking_collection.json` — пример Postman collection.

## Что показывает работодателю

Показывает, что кандидат умеет не только писать код, но и формализовать проверки: предусловия, шаги, ожидаемый результат, приоритет, серьёзность, API-запросы.
