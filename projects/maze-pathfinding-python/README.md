# Maze Pathfinding Python

Алгоритмический проект для поиска пути в текстовом лабиринте. Проект показывает работу с графовым представлением лабиринта, очередями, эвристикой и сохранением результата в файл.

## Возможности

- чтение лабиринта из `.txt`-файла;
- поиск пути до ключа;
- поиск пути до выхода;
- использование BFS и A*;
- сохранение лабиринта с отмеченным маршрутом.

## Стек

Python, heapq, collections, argparse, pathlib.

## Запуск

```bash
python maze_solver.py --input data/sample_maze.txt --output results/solved_maze.txt
```

## Что показывает проект

Проект демонстрирует базовые алгоритмы поиска пути, работу с файлами, структурирование кода и CLI-интерфейс.
