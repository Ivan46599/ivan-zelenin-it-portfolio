# Maze Pathfinding Python

Рефакторинг учебной работы с лабиринтом. Проект решает задачу поиска пути: сначала от аватара до ключа, затем от ключа до выхода.

## Алгоритмы

- BFS — кратчайший путь в невзвешенном графе от старта до ключа.
- A* — эвристический поиск пути от ключа до выхода с манхэттенской эвристикой.

## Запуск

```bash
cd projects/maze-pathfinding-python
python src/maze_pathfinding.py --maze data/sample_maze.txt
```

Для большого исходного лабиринта можно передать координаты явно:

```bash
python src/maze_pathfinding.py --maze data/maze-for-u.txt --start 8,15 --key 2,10 --exit 599,798
```

## Что показывает

Проект демонстрирует структуры данных, графы, работу с файлами, CLI, декомпозицию функций и базовые тесты.
