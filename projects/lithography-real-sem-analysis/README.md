# Lithography Real SEM Analysis

Инженерно-научный проект для анализа реальных SEM/TIF-изображений проявленного фоторезиста. Цель проекта — автоматически выделять линии на изображениях, калибровать масштаб и рассчитывать метрологические параметры CD, LER и LWR.

## Возможности

1. Загрузка SEM/TIF/PNG/JPG-изображений.
2. Удаление служебной области микроскопа и верхних подписей из области анализа.
3. Оценка масштаба `nm/px` по scale bar или использование заданной калибровки.
4. Нормализация контраста SEM-изображения.
5. Поиск тёмных или светлых вертикальных элементов фоторезиста.
6. Извлечение левого и правого края каждой линии по строкам.
7. Расчёт CD, LWR и LER.
8. Сохранение CSV-таблиц и диагностических графиков.

## Стек

Python, NumPy, SciPy, Pillow, matplotlib, dataclasses, argparse. Дополнительно приложена MATLAB-реализация.

## Запуск Python-версии

```bash
pip install -r requirements.txt
python python/analyze_real_resist_images.py --input-dir input_images --results-dir results --figures-dir figures
```

## Структура

```text
python/        # Python-реализация анализа
matlab/        # MATLAB-аналоги функций
input_images/  # SEM/TIF-изображения
results/       # CSV-таблицы с рассчитанными метриками
figures/       # диагностические графики
```

## Результат

На выходе формируются таблицы с рассчитанными метриками и диагностические изображения, которые позволяют визуально проверить выделение линий и корректность расчётов.

## Ограничения

NILS не рассчитывается по SEM-изображению, потому что SEM после проявления не является оптическим intensity profile. NILS корректно считать по optical/air-image intensity profile или соответствующей симуляции.
