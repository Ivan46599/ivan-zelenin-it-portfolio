# C# WinForms Reference

Папка содержит примеры учебных C# WinForms-приложений. Эти проекты используются как дополнительный материал и показывают опыт работы с desktop UI.

## Что входит

- формы и обработчики событий;
- базовая логика desktop-приложений;
- работа с пользовательским вводом;
- примеры интерфейса на WinForms.

## Стек

C#, .NET Framework / WinForms, Visual Studio.

## Что показывает проект

Проект дополняет основное Python/SQL-портфолио и показывает, что есть опыт не только со скриптами, но и с desktop-приложениями.
