# C# WinForms apps

Дополнительные проекты, если откликаться на C#/.NET или desktop-направления.

Содержимое:
- банковское WinForms-приложение с классами клиента/сотрудника/операций;
- WinForms-приложение для работы с водяным знаком в изображении.

Перед публикацией:
- проверить сборку в Visual Studio;
- добавить `.sln`/`.csproj` в чистом виде без `bin`, `obj`, `.vs`;
- переименовать классы и методы в единый стиль;
- добавить скриншоты интерфейса и README.
