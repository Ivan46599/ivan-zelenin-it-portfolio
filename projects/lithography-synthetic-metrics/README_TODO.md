# Lithography synthetic metrics

Кандидат в отдельный GitHub-репозиторий.

Содержит Python/MATLAB-код для генерации модельной структуры фоторезиста и расчета метрологических параметров: CD, LER, LWR и NILS по модельному профилю интенсивности.

Перед публикацией:
- переименовать файлы в единый стиль;
- добавить `requirements.txt`;
- описать математическую постановку;
- оставить 2-4 диагностических графика в `figures/`;
- проверить, что запуск из корня репозитория создает `data/metrics.csv`.
