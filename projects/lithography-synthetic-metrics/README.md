# Lithography Synthetic Metrics

Проект моделирует синтетическую структуру фоторезиста и рассчитывает метрологические параметры литографии: CD, LER, LWR и NILS.

## Что делает проект

- генерирует модель высотной карты фоторезиста;
- задаёт коррелированную шероховатость краёв;
- извлекает левый и правый край линии;
- рассчитывает CD, LER и LWR;
- формирует модельный профиль интенсивности;
- рассчитывает NILS;
- сохраняет графики и CSV-таблицу с результатами.

## Стек

Python, NumPy, matplotlib, dataclasses; дополнительно есть MATLAB-версия алгоритма.

## Запуск

```bash
pip install numpy matplotlib
python main_resist_analysis.py
```

Результаты сохраняются в папки `figures/` и `data/`.

## Что показывает

Проект полезен для стажировок в направлениях scientific computing, data analysis, engineering software, image/data processing: есть численное моделирование, метрики, визуализация и инженерная предметная область.
