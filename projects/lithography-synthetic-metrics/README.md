# Lithography Synthetic Metrics

Проект моделирует синтетическую структуру фоторезиста и рассчитывает метрологические параметры литографии: CD, LER, LWR и NILS.

## Возможности

- генерация модельной карты высоты фоторезиста;
- задание коррелированной шероховатости краёв;
- извлечение левого и правого края линии;
- расчёт CD, LER и LWR;
- формирование модельного профиля интенсивности;
- расчёт NILS;
- сохранение графиков и CSV-таблицы с результатами.

## Стек

Python, NumPy, matplotlib, dataclasses. Дополнительно есть MATLAB-версия алгоритма.

## Запуск

```bash
pip install -r requirements.txt
python main_resist_analysis.py
```

Результаты сохраняются в папки `figures/` и `data/`.

## Что показывает проект

Проект демонстрирует навыки численного моделирования, расчёта инженерных метрик, визуализации данных и работы с предметной областью литографии.
