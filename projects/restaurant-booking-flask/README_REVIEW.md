# Restaurant booking web app

Кандидат в портфолио после доработки.

Что есть:
- Flask-приложение;
- страницы меню, галереи, контактов, бронирования;
- административная страница бронирований;
- MySQL-запросы через Flask-MySQLdb;
- шаблоны Jinja2 и статические файлы.

Что нужно исправить перед GitHub:

- добавить полноценный `schema.sql` и seed-данные;
- убрать хардкод паролей и секретов;
- добавить инструкции запуска;
- проверить соответствие таблиц, которые использует `app.py`, фактической схеме БД.
