# Restaurant Booking Flask App

Учебное web-приложение ресторана: меню, галерея, контакты, бронирование столиков и административный просмотр броней.

## Стек

Python, Flask, Jinja2, MySQL, HTML, CSS, JavaScript.

## Возможности

- главная страница с популярными блюдами;
- меню по категориям;
- галерея;
- форма обратной связи;
- бронирование столиков;
- автоматическая очистка истёкших броней;
- административный просмотр бронирований.

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
flask --app app run --debug
```

## Что показывает

Проект показывает базовый full-stack: Flask routes, templates, формы, SQL-запросы, конфигурация, работа с базой и web-интерфейсом.
