# Restaurant Booking Flask App

Учебное web-приложение ресторана: меню, галерея, контакты, бронирование столиков и административный просмотр броней.

## Стек

Python, Flask, Jinja2, MySQL, HTML, CSS, JavaScript.

## Возможности

- главная страница с популярными блюдами;
- меню по категориям;
- галерея;
- форма обратной связи;
- бронирование столиков;
- автоматическая очистка истёкших броней;
- административный просмотр бронирований.

## Безопасность и конфигурация

Секреты и пароль БД не должны храниться в GitHub. Используй `.env` по примеру `.env.example`.

```bash
cp .env.example .env
```

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
flask --app app run --debug
```

## Что нужно доработать перед публичной демонстрацией

1. Привести схему БД к одному варианту: MySQL или SQLite.
2. Добавить `schema.sql` и `seed.sql` для воспроизводимого запуска.
3. Перевести названия таблиц на латиницу, если проект будет показываться международному работодателю.
4. Добавить валидацию форм и обработку ошибок.
5. Добавить скриншоты интерфейса в README.

## Что показывает работодателю

Проект показывает базовый full-stack: Flask routes, templates, формы, SQL-запросы, конфигурация, работа с базой и web-интерфейсом.
