# SQL Airline Queries

Учебный набор SQL-запросов по данным авиаперелётов: клиенты, билеты, рейсы и авиалайнеры.

## Состав

- `requests.sql` — SQL-запросы;
- `airliners.txt`, `clients.txt`, `tickets.txt`, `trips.txt` — исходные таблицы/данные в текстовом формате.

## Что показывает

Проект полезен как дополнительное подтверждение базовых навыков SQL: выборки, фильтрация, JOIN, агрегации и работа с несколькими связанными таблицами.
