# SQL Airline Queries

Учебный SQL-проект с запросами к базе авиаперевозок.

## Темы запросов

- выборки с фильтрацией;
- сортировка и ограничение результата;
- JOIN между таблицами;
- группировки и агрегатные функции;
- подзапросы;
- аналитические выборки по рейсам, аэропортам и пассажирам.

## Стек

SQL, реляционные базы данных.

## Что показывает проект

Проект демонстрирует базовые навыки работы с реляционными данными и написания SQL-запросов, полезные для backend, QA и data-направлений.
