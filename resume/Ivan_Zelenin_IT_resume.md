## Проекты

### Lithography Real SEM Analysis
**Стек:** Python, NumPy, SciPy, Pillow, matplotlib, MATLAB  
Инженерно-научный проект для анализа реальных SEM/TIF-изображений фоторезиста. Реализованы загрузка изображений, удаление служебных областей, оценка масштаба по scale bar, нормализация контраста, выделение краёв линий и расчёт CD, LER, LWR. Результаты сохраняются в CSV, дополнительно формируются диагностические графики.

### Lithography Synthetic Metrics
**Стек:** Python, NumPy, matplotlib, MATLAB  
Модельная генерация структуры фоторезиста с коррелированной шероховатостью краёв. Реализован расчёт CD, LER, LWR и NILS, построение графиков высотной карты, профиля интенсивности, ширины вдоль линии и распределения ширин.

### Restaurant Booking Flask App
**Стек:** Python, Flask, Jinja2, MySQL, HTML/CSS/JS  
Web-приложение ресторана: меню, галерея, контакты, бронирование столиков и административный просмотр броней. В проекте есть маршруты Flask, шаблоны Jinja2, SQL-запросы, конфигурация через переменные окружения и базовая структура full-stack приложения.

### Maze Pathfinding Python
**Стек:** Python, алгоритмы, BFS, A*, argparse  
Рефакторинг учебного проекта по поиску пути в лабиринте: чтение лабиринта из файла, поиск пути от старта до ключа через BFS и от ключа до выхода через A*. Добавлен CLI, декомпозиция функций и тест на sample-maze.

### Student Performance Analysis
**Стек:** Python, pandas, matplotlib  
Мини-проект по анализу учебных результатов: загрузка CSV, расчёт средних оценок, определение группы риска, группировка по учебным группам и построение графиков.

### API Data Parser
**Стек:** Python, requests, JSON, CSV, argparse  
CLI-утилита для загрузки JSON-данных из файла или API, нормализации вакансий/записей, фильтрации и экспорта в CSV.
